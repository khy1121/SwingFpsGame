# NetFps 주요 변경 사항 요약 (2025-11-29)

## 1. 라운드 시스템 업데이트
- 3판 2선승 방식(최대 3라운드, 2승 팀이 최종 승리) 적용
- 각 라운드 종료 시 승리 팀, 현재 스코어를 브로드캐스트
- 라운드 종료 후 자동으로 다음 라운드 시작(3초 대기)
- 게임 종료 시 `GAME_OVER` 메시지로 클라이언트에 알림

## 2. 맵3 추가 및 맵 랜덤 선택
- 서버에서 라운드 시작 시 맵을 랜덤으로 선택: `map`, `map2`, `map3`, `village` 중 하나가 자동으로 결정됨
- 클라이언트/로비에서 맵3 선택 가능

## 3. 게임 종료 시 로비 복귀
- 한 팀이 2승 달성 시 `GAME_OVER` 메시지 전송
- 클라이언트는 해당 신호를 받아 로비로 자동 복귀

## 4. 캐릭터 변경 제한
- 라운드 시작 후 10초 이내에만 캐릭터 변경 가능
- 라운드당 1회만 변경 허용(중복 변경 방지)

## 5. 기타 서버 기능
- 설치형 오브젝트(지뢰, 터렛) 관리 및 데미지 처리
- 오라 버프, 에어스트라이크, 터렛 자동 공격 등 각종 스킬 효과 서버에서 관리
- 팀/레디/캐릭터 정보, 채팅, 스탯 실시간 브로드캐스트

---

**요약:**
- 라운드 시스템, 맵3, 게임 종료 후 로비 복귀 등 주요 기능이 모두 구현됨
- 서버가 라운드/맵/승리/게임 상태를 관리하며, 클라이언트는 신호에 따라 UI 및 게임 상태를 자동 전환
- 캐릭터 변경 제한, 오브젝트/스킬 효과 등도 서버에서 일관성 있게 처리
